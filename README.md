# Descrição do projeto

Este é um projeto pessoal feito em conjunto com minha namorada onde tivemos a ideia de criar um gerenciador de tarefas. Onde o Usuario colocará o seu perfil de usuario e poderá cadastrar uma tarefa e ter o controle total das 
mesmas, definindo uma prioriodade para cada tarefa e controlado se está em andamento ou não. Possuindo um sistema de movimentações e historico de tarefas feitas e realizadas.

- Utilizamos para o BackEnd a  linguagem Java, com tecnologias como: Spark e Gson para requisições HTTP
- Para o frontEnd Feito pela minha namorada utilizando o React.
